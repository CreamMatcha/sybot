# Sybot

서윤봇 스크립트 저장소

게임 기능과 정보 조회, 자동 알림 기능을 포함합니다.

---

## 구조

sybot/
├─ core-1.0.0/ # 봇 코어
├─ fishingGame/ # 낚시 / 상어 게임
├─ lostark/ # 로스트아크 기능
├─ data/ # JSON 데이터
├─ package.json
└─ .gitignore


---

## 기능

- 낚시 / 상어 게임
- 로스트아크 전정실 조회
- 로펙 조회
---

## 작업 규칙

- `main` 브랜치는 직접 수정하지 않습니다.
- 기능 개발은 브랜치 생성 후 진행합니다.
- 작업 완료 후 Pull Request로 병합합니다.
