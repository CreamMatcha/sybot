# Sybot

서윤봇 스크립트 저장소

서윤봇은 Android GraalJS 런타임에서 동작하는 고성능 메신저 자동화 솔루션으로, 로스트아크 API와 커스텀 클라우드 서버를 활용한 실시간 정보 조회를 제공합니다. 
비동기 처리(async/await)와 모듈화된 설계를 통해 확장성을 극대화한 서윤봇의 통합 개발 환경을 지향합니다.

게임 기능과 정보 조회 등의 기능을 포함합니다.
봇과 연동중인 구글 클라우드 서버(gcp)의 데이터를 포함합니다.

## 기능

- 낚시 / 상어 게임
- 로스트아크 전정실 조회
- 로펙 조회
